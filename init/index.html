2020-07-11

I don't have many plans for this space. But I've been wanting to write more and record some of the stuff I'm figuring out and ultimately forgetting.

Current plans starting with my vacation (July 2020), which happens to coincide with my 40th birthday, is to go fly a kite. But also in the evenings sit down and dig into the The Machinery.

I was given access to the beta after filling out the requisite forms etc. This is pretty exciting for me. It's a unique tool and a unique team of people working on it.

Folks that know me, know that I really went all in with Rust (currently using it at my day job) and The Machinery is built and designed to be used primarily using C.

This sets the stage for not only diving into the world of "modern" C but it gives us that hook for more easily using Rust for some things.
